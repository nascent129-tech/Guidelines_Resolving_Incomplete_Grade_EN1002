1 คำชี้แจงเบื้องต้น

ทำไมนิสิตถึงได้รับเกรด I?

นิสิตที่ได้รับ เกรด I (Incomplete) ในรายวิชานี้ มีสาเหตุมาจาก คะแนนรวมทั้งหมด (คะแนนเก็บและคะแนนสอบ) ยังไม่ถึงเกณฑ์ผ่านรายวิชา

สาเหตุอาจเกิดจากหลายปัจจัย เช่น

  • ไม่ได้เข้าเรียนวิดีโอครบ 15 บท จึงไม่มีคะแนนเก็บในส่วนนั้น
  • ไม่ได้ทำ Pre-test / Post-test ครบทุกบท
  • ไม่ได้เข้าเรียน คลาสสดทั้ง 3 ครั้ง หรือไม่ได้เรียนย้อนหลัง
  • หรือมี คะแนนสอบค่อนข้างน้อย เมื่อรวมกับคะแนนเก็บที่มีอยู่ คะแนนรวมจึงยังไม่ถึงเกณฑ์ผ่าน

เพื่อเปิดโอกาสให้นิสิต สามารถผ่านรายวิชาได้โดยไม่ต้องได้รับเกรด F และไม่ต้องลงทะเบียนเรียนใหม่

อาจารย์จึงเปิดโอกาสให้นิสิตที่ได้รับเกรด I สามารถ ทำคะแนนเพิ่มเติมเพื่อให้ถึงเกณฑ์ผ่าน

2 วิธีการ

วิธีทำแบบฝึกหัดเพื่อได้เกรด D หรือ D+

นิสิตส่วนใหญ่จะสามารถแก้เกรดได้ในระดับ D หรือ D+ โดยการทำ แบบฝึกหัดที่จัดเตรียมไว้ให้ (จะมีลิงก์ให้เข้าทำแบบฝึกหัด โดยอาจารย์จะส่งลิงก์ลงในกลุ่มไลน์)

แบบฝึกหัดในระบบมีจำนวนหลายข้อ และนิสิต ไม่จำเป็นต้องทำทุกข้อ แต่ต้องทำให้ได้จำนวนข้อถูก ตามคะแนนที่อาจารย์แจ้งสำหรับแต่ละคน

นิสิตสามารถทำแบบฝึกหัด มากกว่าจำนวนขั้นต่ำ

3 เงื่อนไขพิเศษ

เงื่อนไขเพิ่มเติมสำหรับผู้ที่ต้องการเกรด C

สำหรับนิสิตที่มี คะแนนเดิมค่อนข้างสูงและใกล้ถึงเกณฑ์ผ่านอยู่แล้ว อาจมีโอกาสแก้เกรดได้ถึงระดับ C

อย่างไรก็ตาม การได้เกรด C ไม่สามารถใช้การทำแบบฝึกหัดเพียงอย่างเดียวได้

นิสิตจะต้องดำเนินการดังนี้

  1. ต้องทำแบบฝึกหัดให้ได้คะแนน อย่างน้อยในระดับ D+ ก่อน
  2. จัดทำ วิดีโออธิบายหลักไวยากรณ์ภาษาอังกฤษ 1 หัวข้อ จากเนื้อหาที่เรียนในรายวิชานี้

เงื่อนไขของวิดีโอ

  • ✓ ต้องเป็น การอธิบายด้วยเสียงของตนเอง
  • ✓ ต้อง เปิดกล้องขณะอธิบาย
  • ✓ ต้องมี สื่อประกอบการอธิบาย เช่น สไลด์ เอกสาร หรือ ตัวอย่างประโยค
  • ✓ ความยาววิดีโอ ไม่น้อยกว่า 3 นาที
  • ✓ วิดีโอควรแสดงให้เห็นว่านิสิต เข้าใจหลักไวยากรณ์และสามารถอธิบายให้ผู้อื่นเข้าใจได้
⚠️ โปรดทราบ
ทั้งนี้ การส่งวิดีโอไม่ได้รับประกันว่าจะได้เกรด C
อาจารย์จะพิจารณาจาก ความถูกต้อง ความครบถ้วน และคุณภาพของการอธิบายอีกครั้ง
